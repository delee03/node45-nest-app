Flow of working

1. User upload image -> handle send it to Bucket in S3
2. Handle API gửi ảnh từ S3 trong Bubket đến Rekognition
3. Response từ Rekognition dùng Lambda để nhận data

Nguồn ảnh: 
-Có thể search các từ khóa ảnh lỗi khi chuyển khoản
- hoặc tự design dữ liệu giả từ canva hoặc figma...


Ví dụ về lỗi cụ thể: 
Tình huống 1: Nhận diện lỗi màn hình xanh (Blue Screen of Death - BSOD)
Dữ liệu giả:

Hình ảnh về màn hình xanh chứa thông báo lỗi như:
css
Copy code
A problem has been detected and Windows has been shut down.
Kết quả từ Rekognition:

Nhãn (Labels): ["Blue Screen", "Error"]
Văn bản (Text): "A problem has been detected"
Phản hồi chatbot:

"Lỗi màn hình xanh xuất hiện khi hệ điều hành gặp lỗi nghiêm trọng. Hãy thử khởi động lại máy tính."






Pricing: 
1. Dịch vụ AWS bạn sẽ sử dụng
Amazon Rekognition:

Tính năng chính: DetectLabels (nhận diện đối tượng), DetectText (nhận diện văn bản).

Chi phí:

Bậc miễn phí: 5.000 ảnh đầu tiên được phân tích mỗi tháng.
Sau bậc miễn phí:
$1.00 / 1.000 ảnh cho DetectLabels.
$0.001 / 1 hình ảnh cho DetectText.
Ước tính: Với 20 ảnh/ngày (600 ảnh/tháng), bạn sẽ nằm trong bậc miễn phí.

Amazon S3:

Tính năng chính: Lưu trữ và truy xuất ảnh người dùng tải lên.
Chi phí:
Lưu trữ dữ liệu: $0.023 / GB mỗi tháng cho lưu trữ chuẩn (Standard).
Truy xuất dữ liệu: $0.005 / GB.
Ước tính:
Nếu mỗi ảnh ~500 KB, lưu trữ 600 ảnh/tháng = ~300 MB = $0.01/tháng.
Truy xuất ảnh 600 lần = < $0.01/tháng.
AWS Lambda:

Tính năng chính: Xử lý logic (tải ảnh lên S3, gọi Rekognition, trả về kết quả).
Chi phí:
Bậc miễn phí: 1 triệu yêu cầu và 400.000 GB-giây mỗi tháng.
Sau bậc miễn phí: $0.20 / 1 triệu yêu cầu.
Ước tính: Với ~20 yêu cầu/ngày (600 yêu cầu/tháng), bạn sẽ nằm trong bậc miễn phí.
API Gateway:

Tính năng chính: Xử lý yêu cầu từ chatbot đến AWS Lambda.
Chi phí:
Bậc miễn phí: 1 triệu yêu cầu/tháng.
Sau bậc miễn phí: $3.50 / triệu yêu cầu.
Ước tính: Với 600 yêu cầu/tháng, bạn nằm trong bậc miễn phí.
Amazon Comprehend (tuỳ chọn):

Tính năng chính: Phân tích ngữ cảnh từ nội dung mô tả lỗi.
Chi phí:
Bậc miễn phí: 50.000 văn bản/tháng.
Sau bậc miễn phí: $0.0001 / 1 văn bản (<=1 KB).
Ước tính: Với 20 mô tả lỗi/ngày (600 mô tả/tháng), bạn nằm trong bậc miễn phí.
2. Tổng chi phí ước tính
Dịch vụ	Chi phí hàng tháng
Amazon Rekognition	Miễn phí (5.000 ảnh/tháng đầu tiên)
Amazon S3	~$0.01 (dung lượng lưu trữ và truy xuất)
AWS Lambda	Miễn phí (1 triệu yêu cầu/tháng đầu tiên)
API Gateway	Miễn phí (1 triệu yêu cầu/tháng đầu tiên)
Amazon Comprehend	Miễn phí (50.000 văn bản/tháng đầu tiên)
Tổng chi phí ước tính: <$0.02/tháng
Bạn gần như sẽ không tốn chi phí nếu nằm trong bậc miễn phí.

3. Nếu vượt quá bậc miễn phí
Nếu bạn vượt bậc miễn phí (rất khó với khối lượng nhỏ):
Rekognition: $1.00 / 1.000 ảnh.
S3: $0.023 / GB lưu trữ.
Lambda/API Gateway: Chi phí thấp, gần như không đáng kể với khối lượng này.
4. Mẹo tối ưu chi phí
Kiểm soát khối lượng ảnh và dữ liệu: Chỉ tải ảnh cần thiết và xóa ảnh không cần thiết để giảm dung lượng lưu trữ.
Sử dụng AWS Free Tier: Theo dõi khối lượng sử dụng trong bảng điều khiển AWS để không vượt bậc miễn phí.
